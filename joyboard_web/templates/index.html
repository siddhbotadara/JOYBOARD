What is JOYBOARD?

Experience gaming like never before — where software meets hardware.

JOYBOARD is a fully integrated gaming ecosystem that merges fast-paced 2D platforming with cutting-edge technology. At its core, it combines an exhilarating 2D platformer, a real-time web analytics dashboard, and a custom-built Arduino wireless controller — delivering an experience that's interactive, tangible, and uniquely engaging.

Players navigate intricately designed levels filled with smart, dynamic challenges while competing on live leaderboards that track performance in real time. The web dashboard offers detailed gameplay insights, empowering players to analyze strategies, improve skills, and engage more deeply with every session.

With seamless hardware-software integration, JOYBOARD transcends traditional gaming. It creates an immersive, competitive ecosystem that blends physical interaction, digital analytics, and innovative gameplay — redefining how players experience and connect with games.
