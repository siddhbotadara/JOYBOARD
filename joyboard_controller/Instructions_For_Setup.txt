üéÆ SETTING UP YOUR ARDUINO CONTROLLER (TRANSMITTER)

You can build a simple wireless transmitter using an Arduino Nano or Uno board.

üîß Required Components (Minimum):
- 2x Joysticks

‚ö†Ô∏è **Important Note on NRF24L01 Power Supply:**

 >> Arduino 5V ‚Üí Voltage Divider ‚Üí NRF VCC (3.3V)

üì° SETTING UP THE RECEIVER
------------------------------------------------------------------------------------------------------------------------

Use another Arduino Nano or Uno as your receiver. Connect only the NRF24L01 module to the following pins:

| NRF24L01 Pin | Arduino Pin |
|--------------|--------------|
| CE           | D9           |
| CSN          | D10          |
| MOSI         | D11          |
| MISO         | D12          |
| SCK          | D13          |

No other components are needed on the receiver side.

------------------------------------------------------------------------------------------------------------------------
üíæ UPLOADING THE CODE
------------------------------------------------------------------------------------------------------------------------

1. Install the official Arduino IDE from the Arduino website - https://www.arduino.cc/en/software
2. Open the transmitter and receiver `.ino` files located in the `arduino_files` folder.
3. Select the correct board (Uno/Nano), COM port, and set the **baudrate to 9600**.
4. Upload the **transmitter** code to your controller board.
5. Upload the **receiver** code to your receiver board.
6. Open the Serial Monitor from the Arduino IDE (at 9600 baud) and confirm if data is being received correctly.
7. Once confirmed, **close the Arduino IDE completely**. 
8. Open the arduino_setup.txt file located in the same folder and update the port and baud rate used.
9. Close the file and launch the game.
